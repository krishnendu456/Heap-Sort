# Heap Sort in Python

This project implements the Heap Sort algorithm in Python. Heap Sort is an efficient comparison-based sorting technique based on the Binary Heap data structure.

# Features:
Simple and clean implementation
In-place sorting (no extra memory required)
Efficient for large datasets
Time complexity: O(n log n)

# How It Works:
Build a Max Heap from the input array.
Swap the root (largest element) with the last element.
Reduce the heap size.
Heapify the root again.
Repeat until the array is sorted.
